Python for Teens

If you are between 14 and 18 years old, this course is for you! You will learn the basics of the python language in a pleasant and structured way, with straightforward explanations and fun examples. You will also learn computational and logical thinking, which is the foundation of any code you will write in the future.

You will learn:

  • Data types: Strings, lists, and dictionaries
  • Conditions: if / elif / else
  • Loops: for and while
  • Built-in functions: input(), print(), and more
  • Python packages: random, os, and more
  • Games: Rock-paper-scissor, and more

Python for Beginners

Either you are a university student or a professional, this course is for you! It will provide you the right training to reach your academic or professional goals. You will learn python programming in a structured and easy way, so that you will create solid basics for coding and computational thinking.

You will learn:

  • Data types: Strings, lists, and dictionaries
  • Conditions: if / elif / else
  • Loops: for and while
  • Built-in functions: input(), print(), and more
  • Python packages: random, os, and more
  • Projects: Create username, password, and more
